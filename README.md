## About this site

Built deliberately as a **single self-contained `index.html`** — no framework, no build step,
no dependencies. All styles, scripts, and icons are inline. Clone it and open the file, and
it works offline.

- Light and dark themes, remembering your choice and following the system default
- Filterable project list (Salesforce / Full-Stack / AI & Automation)
- Progressive disclosure — technical detail is collapsed until you want it
- Graceful degradation: any missing screenshot or PDF simply hides instead of breaking
- Accessible markup with ARIA labels, and a print stylesheet that auto-expands every project for Ctrl+P → Save as PDF

### Run it locally

```bash
git clone https://github.com/shailesh120/ShaileshPortfolio.git
```

Then open `index.html` in any browser. That's the whole setup — there is nothing to install.
